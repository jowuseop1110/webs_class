조운섭

CSS padding

padding

padding 속성은 네 방향 안쪽 여백 영역을 설정합니다.

문법

padding: length ㅣ & ㅣ auto

정의

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

    • 예제

padding-top: 10px;                  /* 위 10px
padding-right: 11px;                /* 오른쪽 11px
padding-bottom: 12px;               /* 아래쪽 12px
padding-left: 13px;                 /* 왼쪽 13px
padding: 10px;                               /* 위,오른쪽,아래,왼쪽 : 10px */
padding: 10px 11px;                      /* 위,아래:10px 왼쪽,오른쪽:11px */
padding: 10px 11px 12px;            /* 위:10px 왼쪽,오른쪽:11px 아래:12px */
padding: 10px 11px 12px 13px;  /* 위:10px 오른쪽:11px 아래:12px 왼쪽:13px */
