기타 애니메이션 효과 테이블
전체 속성 갯수 : 0개
  • all : all 속성은 CSS 속성을 재설정하는 데 사용할 수 있는 약식 속성입니다.
  • animation : animation 속성은 애니메이션 속성을 설정하기 위한 약식 속성입니다.
  • animation-delay :animation-delay 속성은 애니메이션이 시작되는시간을 설정합니다.
  • animation-directio :animation-direction 속성은애니메이션이 일부 또는 모든 주기 또는 반복에서 반대로재생되어야 하는지 여부를 지정합니다.
  • animation-duratio :animation-duration 속성은 애니메이션 주기에 걸리는 시간을 지정하는 데 사용됩니다.
  • animation-fill-mode : animation-fill-mode 속성은 애니메이션이 끝난 후의 상태를 설정합니다.
  • animation-iteration-count : animation-iteration-count 속성은 애니메이션이 중지되기 전에 애니메이션 주기가 재생되는 횟수를 지정하는 데 사용됩니다.
  • animation-name : animation-name 속성은 @keyframes 규칙에의해 정의된 하나 이상의 애니메이션 이름을 지정하는데 사용되며 선택한 요소에 적용됩니다.
  • animation-play-state : animation-play-state 속성은 CSS 애니메이션이 실행 중인지 일시 중지되었는지 지정합니다.
  • background-filter : background-filter 속성을 사용하면 CSS를 사용하여 요소 뒤의 콘텐츠에 필터 효과를 적용할 수 있습니다.
  • backface-visibility : backface-visibility 속성은 뷰어를 향할 때 변형된 요소의 "뒷면"이 보이는지 여부를 결정합니다.
  • caption-side : caption-side 속성은 테이블 caption의 위치를 설정합니다.
  • direction : direction 속성은 텍스트 흐름의 방향을 제어하는​​데 사용됩니다.
  • display : display 속성은 페이지의 요소에 대해 생성되는 상자의 종류를 지정하는 데 사용됩니다.
