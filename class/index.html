1. 학습개요

훈련시간 168시간 담당교사 　　　　
훈련개요 UI 설계 산출물과 Gui 디자인 다이드를 바탕으로 UI 구현 표준을 검토하는 법
을 학습한다.
훈련목표 UI 설계 산출물과 GUI 디자인 가이드를 바탕으로 UI 설계와 UI 구현 펴준을 검
토할 수 있다. 을 학습한다.
세부목표
단원명 단원별 학습목표
2001020708_19v3.1
UI 설계 검토하기		 1.1 UI 제작을 위하여 GUI 디자인 가이드를 이해하고 이를 기반으로 구현 가능성 여
부를 검토할 수 있다.
1.2 구현 환경에 따라서 구체적인 GUI 프로세서의 이해와 설계 변경 여부를 파악할
수 있다. 3.1 UI 구현 표준 수립을 위하여 UI 검토 의견서를 작성할 수 있다.
2001020708_19v3.2
UI 구현 표준
검토하기		 2.1 고객 요구사항, 접근성 기준, 플램폼에 대한 UI 표준 관련 기준을 파악할 수 있
다.
2.2 실제 페이지 제작을 위하여 아이콘, 레이아웃, 화면 개발 환경이 적합한 표준을
검토할 수 있다.
2.3 UI 구현 표준을 이해관계자와 검토하여 최종 표준안에 반영할 수 있다.
