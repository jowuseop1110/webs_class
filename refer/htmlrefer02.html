웹표준

웹 표준(Web Standard)란 요약해서 '웹에서 사용되는 표준 기술이나 규칙'을 의미합니다.

웹 표준 관련 기술
  • 구조(HTML)
  • 표현(CSS)
  • 동작(Script)

웹 표준의 장점

  • 손쉬운 유지 보수 및 비용 절감 효과
  • 웹 접근성 향상
  • 검색엔진의 최적화
  • Fille Size 축소, 서버 저장 공간 절약
  • 호환성 확보
  • 효율적 마크업

웹접근성

보통 사람들과 달리 웹 이용에 불편함을 겪는 사람들(장애인, 노약자 등)은 웹의 접근성이 떨어진다.
이 모든 사람들에게 평등한 웹 이용을 위해, 웹 접근성이 만들어졌다.

웹호환성

웹 호환성은 표준 웹 기술을 사용하여 운영체제, 브라우저 등 어느 한쪽으로 최적화되거나 종속되지 않도록
공통 요소를 사용하여 웹 페이지를 제작하는 기법으로 웹 사이트 사용 시 운영체제 및 브라우저 간 동일한
결과가 나오도록 의미하는 웹 상호운용성의 개념에 웹 표준의 준수를 포함하는 개념이다.

목적 : 웹 브라우저 버전, 종류와 관계없는 웹사이트 접근

준수 내용

  • 웹 표준 준수를 통한 브라우저 호환성 확보
  • HTML, CSS 문법 준수
  • 동작, 레이아웃, 플러그인 호환성
