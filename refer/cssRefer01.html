CSS란?

CSS는 (Cascading Style Sheets)는 웹페이지를 꾸미려고 작성하는 코드입니다.

CSS 주석 표시

/* CSS 주석 */

CSS 선언 방법

color : red와 같은 단일 규칙, 꾸미기 원하는 요소의 속성을 명시하면 됩니다. 

p {
    color: red;
}

내부 스타일

내부 스타일은 내부에 스타일시트를 작성하는 것을 내부 스타일이라고 합니다. 내부스타일은 대개 단발성 페이지의 CSS 분량이 적은 경우에만 사용합니다. 

<html>
<head>
    <style type="text/css">
       h1 {
           color: blue;
           font-size: 16px;
       }            
    </style>
    <body>
        <h1>Emned style Sheet</h1>
    </body>
</html>

외부 스타일

외부에 스타일 시트를 작성하는 것을 말합니다. 가장 권장되고 일방적인 방법으로 CSS 파일을 별도로 관리하는 형태입니다. 

<html>
   <head>
      <link rel="stylelesheet" href="css/style.css" type="text/css">
   </head>
   <body>
        <h1>Embed style sheet</h1>
   </body>
</html>

인라인 스타일

한줄(라인)로 요소 내부에 스타일 속성을 이용하여 CSS 코드를 작성하는 것 을 인라인 스타일 이라고 합니다. 

<h1 style="color: blue; fonr-size: 16px;">
         Embed style Sheets
</html>
